# IL2CPP MOD LOADER
IL2CPP mod loader is a work-in-progress build-in mod loader for il2cpp games. currently only for android, but ios is being developed this was originally created for the game infinitebox.

NOTE: THE IL2CPP ASSEMBLY GENERATOR IS NOT INCLUDED. you may use my [generator script](https://github.com/MelvinShwuaner/Il2CppAssemblyGenerator)

## PROGRESS:
Host CORECLR and run EntryPoint: Complete, almost ready on IOS

run il2cppinterop: Complete

load plugins: Ready, needs more work
