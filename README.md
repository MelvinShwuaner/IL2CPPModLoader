# IL2CPP MOD LOADER
IL2CPP mod loader is a work-in-progress build-in mod loader for il2cpp games. currently only for android. this was originally created for the game infinitebox.

NOTE: THE IL2CPP GENERATOR IS NOT INCLUDED. i will be working on a built-in generator which runs when your game builds. for now, you may use my [generator script](https://github.com/MelvinShwuaner/Il2CppAssemblyGenerator)

## PROGRESS:
Host CORECLR and run EntryPoint: Complete

run il2cppinterop: Complete

load plugins: Ready, needs more work
